fetch_end_voltage(id)¶

Function to fetch the end voltage. the table is pivoted, ie in the rows there are the cycles, the previous step column is the type of the previous step and the values are the time duration of the given step type as a function of the cycle the table contains the last voltage measurement of the rest step and it reports if the previous step is a charge or discharge step.

Parameters:

id (str) – cell id, e.g. ‘1003-BQV000000000000217-1’.

Returns:

df – table with the end voltage.

Return type:

pd dataframe
